O Surgimento da Internet

(Voltar)

A história da Internet começou em 1969 com a criação da ARPANET, um projeto do Departamento de Defesa dos Estados Unidos. Seu objetivo inicial era conectar universidades e centros de pesquisa, permitindo comunicação mesmo em caso de ataques. A rede usava a comutação de pacotes, uma inovação para a época, e foi considerada uma resposta à corrida tecnológica da Guerra Fria. Os primeiros testes envolveram instituições como a UCLA e o Instituto de Pesquisa de Stanford, marcando o início da troca de informações entre computadores à distância.

Com o crescimento da rede, foi necessário desenvolver padrões para a comunicação entre diferentes sistemas. Nos anos 1970 e 1980, surgiram os protocolos TCP/IP, que se tornaram a base da comunicação na Internet. Em 1983, a ARPANET adotou oficialmente esse protocolo, marcando o nascimento da Internet moderna. Nos anos seguintes, universidades e centros de pesquisa de outros países também passaram a se conectar à rede. Esse período foi essencial para transformar a Internet de uma rede experimental em uma infraestrutura global de troca de dados.

A grande virada aconteceu nos anos 1990, com o surgimento da World Wide Web, criada por Tim Berners-Lee. A Web tornou a Internet mais acessível ao público, com o uso de navegadores e páginas em hipertexto. A popularização da Internet foi impulsionada pelo avanço dos computadores pessoais, da banda larga e, mais tarde, dos smartphones. A partir daí, a rede passou a ser utilizada em áreas como educação, comércio, lazer e comunicação pessoal. Hoje, a Internet é essencial no cotidiano de bilhões de pessoas ao redor do mundo.

Oque e HTML

(Voltar)

HTML (HyperText Markup Language) é a linguagem de marcação fundamental da web, usada para estruturar o conteúdo de uma página, como títulos, parágrafos, listas, imagens e links. Diferente de linguagens de programação, HTML organiza o conteúdo por meio de “elementos” que envolvem o texto com tags de abertura e fechamento, determinando como o navegador deve exibir cada parte. Os elementos podem conter atributos, que fornecem informações adicionais — por exemplo, a tag <img> precisa do atributo src para exibir uma imagem.

A estrutura básica de um documento HTML inclui as seções <head> e <body>. A primeira (head) contém metadados como título, charset e links para CSS ou scripts, essenciais para o funcionamento da página, mas não exibidos diretamente ao usuário. A parte visível (body) reúne os conteúdos principais — texto, imagens, links, etc. — utilizando elementos como <header, <main, <footer, <article> e <section> para organizar semanticamente o documento.

No MDN, o guia "Creating the content" orienta iniciantes a entenderem como criar parágrafos ( <p>), cabeçalhos ( <h1>– <h6>), listas e links ( <a href="…">), além de inserir imagens ( <img src="…">). Ele enfatiza boas práticas como fechar corretamente as tags, usar atributos entre aspas e estruturar o conteúdo usando semântica adequada para melhorar acessibilidade e SEO. Com essa base, o próximo passo no MDN é aprender CSS e JavaScript para estilizar e dar interatividade à página.

O que são tags HTML?

(Voltar)

HTML (HyperText Markup Language) é a linguagem fundamental para estruturar páginas na web. As tags HTML são elementos que definem diferentes partes do conteúdo, como parágrafos, títulos, listas e links. Elas são escritas entre colchetes angulares << > e geralmente aparecem em pares: uma tag de abertura e uma de fechamento. Por exemplo, <p>Este é um parágrafo.</p>

Tipos de tags HTML

  • As tags HTML podem ser divididas em várias categorias, cada uma com uma função específica. Algumas das categorias mais comuns incluem:
  • Ação de texto: <p>, <h1> a <h6>, <ul>, <ol>, <li>, entre outras.
  • Links e imagens: <a> para links e <img> para imagens.
  • Formulários: <form>, <input>, <select>, <textarea>, entre outras.
  • Tabelas e divisões: <table>, <tr>, <td>, <div>, <span>, entre outras.
  • Estilo: <style>, <link>, <meta>, <title>, entre outras.
  • Multimídia: <audio>, <video>.
  • Semânticas: <header>, <footer>, <nav>, <article>, <section>, entre outras.
